<div class="gray-back"><ul><li>1.点击右下角的&#8221;网络&#8221;图标，然后点击&#8221;网络和Internet设置&#8221;-&gt;&#8221;VPN&#8221;-&gt;&#8221;添加VPN连接&#8221;。</li><li>2.进入添加页面之后，需要输入连接名称、服务器IP地址、用户名、密码，然后选择VPN协议。</li><li>3.设置好之后，点击保存退出。然后再点击&#8221;网络&#8221;图标，就可以看到刚刚添加的VPN连接，点击即可使用。</li></ul></div>
